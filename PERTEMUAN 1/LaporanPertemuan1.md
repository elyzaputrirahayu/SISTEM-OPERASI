### 1.10 Latihan 

#### Latihan 1.1
Jelaskan 5 fungsi utama sistem operasi dengan contoh konkret dari minimal 2
OS berbeda (Windows, macOS, atau Linux).
jawab : 
1. Process Management
Mengatur proses yang berjalan dan pembagian CPU.
Contoh:
Windows → Task Manager
Linux → ps, top, htop
2. Memory Management
Mengatur penggunaan RAM dan virtual memory (swap/pagefile).
Contoh:
Windows → pagefile
Linux → swap
3. File Management
Mengatur sistem file dan izin akses.
Contoh:
Windows → NTFS
Linux → ext4
4. I/O Management
Mengatur komunikasi dengan perangkat keras (keyboard, printer, dll).
5. Security & Protection
Melindungi sistem dengan autentikasi & izin akses.
Contoh:
Windows → BitLocker
macOS → FileVault

#### Latihan 1.2 
Kapan sebaiknya menggunakan Windows vs Linux vs macOS? Analisis
berdasarkan use case: gaming, development, server, creative work, dan enterprise.
jawab : Pemilihan sistem operasi sebaiknya disesuaikan dengan kebutuhan pengguna. Untuk kebutuhan gaming, Windows lebih direkomendasikan karena memiliki kompatibilitas game dan dukungan driver yang paling lengkap. Untuk development, Linux atau macOS lebih cocok karena keduanya berbasis Unix dan memiliki dukungan tools pemrograman yang kuat serta terminal yang powerful. Dalam penggunaan sebagai server, Linux menjadi pilihan utama karena stabil, aman, ringan, dan tidak memerlukan biaya lisensi. Untuk pekerjaan creative work seperti desain grafis, editing video, dan produksi musik, macOS sering menjadi standar industri karena optimalisasi hardware dan software yang baik. Sedangkan dalam lingkungan enterprise, Windows Server maupun Linux banyak digunakan karena mendukung manajemen jaringan, keamanan, serta integrasi sistem yang baik sesuai kebutuhan perusahaan.

#### Latihan 1.6
Sehari-hari saya menggunakan sistem operasi Windows. Saya sudah menggunakannya sejak lama, kurang lebih sejak duduk di bangku sekolah dasar hingga sekarang di bangku perkuliahan. Windows menjadi sistem operasi yang paling familiar bagi saya karena hampir semua komputer di sekolah maupun di rumah menggunakan Windows.

Hal yang saya sukai dari Windows adalah tampilannya yang user-friendly dan mudah digunakan, terutama untuk pemula. Selain itu, kompatibilitas software di Windows sangat luas. Banyak aplikasi perkuliahan, seperti Microsoft Office, aplikasi desain, hingga software pemrograman yang dapat berjalan dengan baik di Windows. Windows juga unggul dalam hal gaming dan dukungan driver perangkat keras.

Namun, saya juga pernah mengalami beberapa tantangan saat menggunakan Windows, seperti sistem yang terasa lambat ketika terlalu banyak aplikasi berjalan, update otomatis yang terkadang mengganggu, serta risiko virus jika tidak berhati-hati saat mengunduh file dari internet.

Selain Windows, saya mulai mengenal Linux, khususnya Ubuntu Server, setelah mempelajari materi pada bab ini. Pengalaman menggunakan Linux cukup berbeda karena lebih banyak menggunakan terminal dibandingkan antarmuka grafis. Awalnya terasa sulit dan membingungkan, tetapi lama-kelamaan saya mulai memahami bahwa Linux memberikan kontrol yang lebih besar terhadap sistem. Saya merasa Linux sangat cocok untuk pembelajaran sistem operasi, server, dan administrasi jaringan.

Setelah mempelajari bab ini, saya menjadi tertarik untuk lebih mendalami Linux, terutama karena sistem ini banyak digunakan dalam dunia server dan cloud computing. Selain itu, saya juga penasaran untuk mencoba macOS suatu hari nanti karena terkenal stabil dan banyak digunakan oleh developer serta pekerja kreatif.

Secara keseluruhan, setiap sistem operasi memiliki kelebihan dan kekurangannya masing-masing. Windows nyaman untuk penggunaan sehari-hari, Linux kuat untuk server dan pengembangan, dan macOS unggul dalam integrasi serta pekerjaan kreatif. Dengan memahami perbedaan tersebut, saya dapat memilih sistem operasi yang sesuai dengan kebutuhan saya di masa depan.
